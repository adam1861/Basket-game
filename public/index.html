How to play

  • Allow camera access when prompted.
  • Move your left and right hands to control the bats.
  • Keep the ball in play to rack up points.
  • Press R or the reset button to restart.
  • Quit by closing the tab (desktop legacy key: Q).

Deploy to Vercel

  1. Push this repo to GitHub (or import it into Vercel).
  2. Create a Vercel project and set the output folder to public.
  3. Deploy - the game runs fully client-side.
